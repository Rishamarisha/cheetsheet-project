Typography Cheat Sheet

Item Description
font-weight Used to assign a weight of a font: thin, bold, 700 etc.
font-style italics
line-height Declares vertical spacing between lines of text.
serif/sans-serif adjust the style of the font (edges/no edges)
fallback fonts Add them in font-family, just in case a user doesn't have primary font on their computer.
word-spacing To adjust spacing between words.
letter-spacing To adjust spacing between letters.
text-align used to control how text is aligned within its containing content box (left, right, center, justify).
Linking fonts Linking fonts in HTML via "" tag in head part.
Linking fonts using @font-face in CSS.
